# 📚 Miniguia de Estudos com NotebookLM

## 🚀 Desafio DIO - Aprendendo com IA Generativa

## 📖 Contexto

Este projeto foi desenvolvido como parte do desafio da DIO, utilizando o NotebookLM como ferramenta de aprendizagem ativa.

O tema escolhido foi **Inteligência Artificial Generativa**, com o objetivo de compreender seu funcionamento, aplicações, principais ferramentas e a importância do uso responsável dessa tecnologia.

# 🎯 Objetivos

* Compreender o conceito de Inteligência Artificial Generativa.
* Entender como os modelos generativos funcionam.
* Conhecer as principais ferramentas disponíveis no mercado.
* Explorar aplicações práticas da IA em diferentes áreas.
* Aprender técnicas de Engenharia de Prompts utilizando o NotebookLM.
* Consolidar o conhecimento em um miniguia para futuras consultas.

# 🤖 Engenharia de Prompts

 Prompt 1

**O que é Inteligência Artificial Generativa?**
Resultado

A IA Generativa é uma tecnologia capaz de criar novos conteúdos, como textos, imagens, vídeos, músicas e códigos.

 Prompt 2

**Como funciona a IA Generativa?**
Resultado

Ela aprende padrões a partir de grandes volumes de dados e utiliza esse conhecimento para gerar conteúdos inéditos conforme as instruções do usuário.

Prompt 3

**Explique IA Generativa como se eu fosse um iniciante.**
Resultado

A IA Generativa funciona como um assistente inteligente que transforma ideias em conteúdos diversos utilizando linguagem natural.

Prompt 4

**Quais são as principais aplicações da IA Generativa?**

Resultado

Produtividade, educação, desenvolvimento de software, criação de imagens, vídeos, músicas, pesquisas científicas e saúde.


Prompt 5

**Quais ferramentas utilizam IA Generativa?**

Resultado

Gemini, NotebookLM, ChatGPT, Sora, Veo, Lyria, Gemini Code Assist, Microsoft Azure AI Foundry, entre outras.



 Prompt 6

**Como funciona o treinamento de um modelo generativo?**

 Resultado

Os modelos analisam enormes bases de dados para identificar padrões e gerar novos conteúdos semelhantes aos utilizados durante o treinamento.

Prompt 7

**O que são prompts?**

Resultado

Prompts são instruções fornecidas pelo usuário que orientam a IA sobre qual tarefa executar.

Prompt 8

Por que o uso responsável da IA é importante?

 Resultado

Para garantir segurança, transparência, ética e reduzir riscos como desinformação e uso inadequado da tecnologia.

⚠️ Dificuldades Encontradas

Durante os testes, algumas respostas ficaram muito amplas.

Para obter melhores resultados foi necessário:

* Fazer perguntas mais específicas;
* Solicitar exemplos;
* Pedir respostas resumidas;
* Pedir comparações entre conceitos;
* Reformular prompts para aumentar a precisão.

Esses testes mostraram como a qualidade do prompt influencia diretamente a qualidade das respostas.

# 📖 Glossário

**IA (Inteligência Artificial):** tecnologia que simula capacidades humanas.

**IA Generativa:** IA capaz de criar novos conteúdos.

**Machine Learning:** técnica que permite à IA aprender com dados.

**Deep Learning:** ramo do aprendizado de máquina baseado em redes neurais profundas.

**LLM (Large Language Model):** modelo de linguagem treinado com grandes volumes de texto.

**Prompt:** comando enviado pelo usuário para a IA.

**Engenharia de Prompt:** técnicas para criar instruções que gerem respostas mais precisas.

**GAN (Rede Adversária Generativa):** modelo utilizado para gerar novos dados semelhantes aos dados de treinamento.

**Embeddings:** representação numérica de palavras, imagens e outros dados para facilitar o processamento pela IA.

**Multimodalidade:** capacidade de compreender diferentes tipos de dados, como texto, imagem e áudio.
